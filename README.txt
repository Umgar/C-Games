#POLSKI#

Program jest klonem popularnej gry Flappy Bird. Aplikacja działa w konsoli, a wszystkie elementy graficzne są tworzone za pomocą symboli Unicode.

Program został napisany w języku C na system operacyjny Linux.

Aby skompilować projekt, wymagana jest biblioteka ncurses. Można ją zainstalować za pomocą następującej komendy:
sudo apt-get install libncurses5-dev

Aby zbudować projekt, wystarczy wykonać polecenie:
make all

W rezultacie otrzymamy plik CGame.out.

#ENGLISH#

The program is a clone of the popular game Flappy Bird. The application runs in the console, and all graphical elements are created using Unicode symbols.

The program is written in the C language for the Linux operating system.

To compile the project, the ncurses library is required. You can install it with the following command:
sudo apt-get install libncurses5-dev

To build the project, simply execute the command:
make all

You will then receive the CGame.out file.
